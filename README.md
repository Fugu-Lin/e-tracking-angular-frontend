# e-tracking-angular-frontend

※ 請注意，本專案非官方推出，如需撤除，請告知。

統一超商交貨便貨態查詢前端

此repository與https://github.com/Fugu-Lin/e-tracking-django-backend 為一組全端的專案，可搭配服用。

以下為系統的展示網址:  
http://54.179.188.161:4200/

介面可自由增加輸入貨運編號的欄位，請在"新增選項"上點擊即可。

若輸入的資料沒有帶回，請再重新嘗試一次。
